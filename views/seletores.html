Seletores

Um importande aspecto do CSS é a utilização dos seletores para selecionar qualquer elemento do seu HTML. Com a utilização do SASS, você poderá utilizar os seletores aninhados, sem a a necessidade de repetir seletor para cada elemento que você quer formatar, facilitando e deixando seu código mais simples e escalável. Lembre-se sempre... Boas práticas de CSS devem ser respeitasdas! 

.contato {
  width: 100%;
  height: auto;
  
  .formularioDeContato {
    width: 100%;
    max-width: 450px;
    margin: 0 auto;
    
    input[type="text"] {
      margin: 2px;
      height: 30px;
      width: 100%;
      display: block;
    }
    input[type="submit"] {
      background: #CCCCCC;
      border: none;
      padding: 10px 25px;
    }
  }  
}
