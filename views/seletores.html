Seletores

Um importande aspecto do CSS é a utilização dos seletores para selecionar qualquer elemento do seu HTML. Com a utilização do SASS, você poderá utilizar os seletores aninhados, facilitando e deixando seu código mais simples e escalável.

section {
  width: 100%;
  height: auto;
  
  form {
	width: 450px;
    margin: 0 auto;
    
    input {
      margin: 2px;
      height: 30px;
      width: 100%;
      display: block;
    }        
  }  
}
