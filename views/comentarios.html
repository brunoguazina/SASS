Como no CSS, para você deixar um comentário em seu código basta utilizar /* seu comentario */, no SASS alem dessa sintaxe ja conhecida nas folhas de estilho, você também podera utilizar //seu comentario, a diferenca entre um e outro é que a comentário somente com // nao é compilado para o CSS, permanecendo apenas no SASS. 

// Inicio formulario de contato
.contato {
  width: 100%;
  height: auto;
  
  .formularioDeContato {
    width: 100%; 
    max-width: 450px; /* largura maxima */
    margin: 0 auto; 
    
    input[type="text"] {
      margin: 2px;
      height: 30px;
      width: 100%;
      display: block;
    }
    input[type="submit"] { /* formatacao do botao de enviar formulario */
      background: #CC6699;
      border: none;
      padding: 10px 25px;
      color: #FFFFFF;
    }
  }  
}
// Fim formulario de contato
