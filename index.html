Introdução

Com a evolução da internet nos últimos anos, a complexidade das folhas de estilo devido as interfaces cada vez mais complexas, surgiu os pre-processadores para ajudar.

Criado em 2016 por Hampton Catlin, o SASS é um pré processadores de CSS, que possui diversos recursos(Mixins, Placeholders, Funções, Variáveis, Condicionais, Laços, etc).

Permitem escrever um código mais limpo e escalável, tornar o desenvolvimento de CSS mais ágil e dinâmico, facilitando o reuso e aumentando a produtividade.

Possui uma grande comunidade de desenvolvedres e empresas de tecnologia que tem trabalhado incessantemente para não so manter, mas continuar evoluindo. Você tambem vai encontrar diversos frameworks como: Compass, Bourbon, Susy entre outros.

Hoje em dia, caso não queira utilizar o terminal e não trabalhe com node.js ou Ruby por exemplo... você pode utilizar alguns aplicativos externo como este (Koala) que vão compilar esse codigo para CSS.

SCSS 
$corPadrao: #cc6699;
$tamanhoDaFontePadrao: 14px;

@mixin tipografia($cor: $corPadrao){
  font-family: 'Raleway', sans-serif;
  font-weight: 400;
  color: $cor;
}

$listaDeTitulos: 'h6', 'h5','h4', 'h3', 'h2','h1';
@each $titulo in $listaDeTitulos{
  
  $tamanhoDaFontePadrao: $tamanhoDaFontePadrao + 5;
  #{$titulo}{
    @include tipografia();
    text-transform: uppercase;
    text-align: center;
    font-size: $tamanhoDaFontePadrao;
  }
}
